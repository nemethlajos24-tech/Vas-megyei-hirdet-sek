# Új ügyfél felviteli ellenőrzőlista

## 1. Beküldött adatok ellenőrzése

- [ ] Űrlapbeküldés megérkezett
- [ ] Vállalkozás neve megvan
- [ ] Kapcsolattartó neve megvan
- [ ] Település megvan
- [ ] Kategória megvan
- [ ] Telefonszám megvan
- [ ] Email cím megvan
- [ ] Weboldal vagy Facebook link megvan / nincs megadva jelölve
- [ ] Rövid szolgáltatáscím megvan
- [ ] Rövid főoldali bemutatkozás megvan
- [ ] Hosszabb adatlapos bemutatkozás megvan
- [ ] Fő szolgáltatások megvannak
- [ ] Szolgáltatási terület megvan
- [ ] Nyitvatartás megvan / nincs megadva jelölve
- [ ] Miért érdemes választani megvan
- [ ] Kép / logó megvan vagy helykitöltő szükséges
- [ ] Hozzájárulások rendben

## 2. Fájlok előkészítése

- [ ] Ügyfél munkalap létrehozva az `ugyfelek/` mappában

- [ ] Adatlap fájlnév létrehozva a `vallalkozasok/` mappában
- [ ] Adatlap link ellenőrizve

- [ ] `sablonok/kartya-sablon.html` alapján kártya elkészítve

- [ ] `data-search` kulcsszavak megadva
- [ ] kép megjelenik
- [ ] ikon megfelelő
- [ ] kiemelt jelzés megfelelő
- [ ] telefonszám megjelenik
- [ ] email megjelenik
- [ ] Adatlap gomb jó fájlra mutat

- [ ] Szolgáltatási terület blokk megvan
- [ ] Elérhetőségek blokk jobb oldalon van
- [ ] telefon link működik
- [ ] email link működik
- [ ] weboldal / Facebook link működik
- [ ] kép megfelelően jelenik meg
