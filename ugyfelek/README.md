# Ügyfelek

Ebben a mappában lesznek az egyes vállalkozások munkalapjai.

Minden ügyfélhez külön `.md` fájl készül, amelyben szerepelnek:

- alapadatok
- elérhetőségek
- főoldali kártya adatai
- adatlap adatai
- képfájl neve
- aktuális állapot
- ellenőrzőlista

Ez a mappa belső munkához készül, hogy minden vállalkozás adatai külön, átláthatóan kezelhetők legyenek.
